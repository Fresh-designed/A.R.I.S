Acerca del Sistema
Simulador de Historiales Clínicos

Este es un prototipo de sistema de gestión de historiales clínicos para hospitales.

Importante: Todos los datos generados son sintéticos y no representan a personas reales.

El sistema permite:

  • Visualizar registros de pacientes
  • Buscar y filtrar registros
  • Editar información existente
  • Generar registros sintéticos para pruebas
  • Exportar datos en formato CSV

Almacenamiento: Todos los datos se almacenan localmente en el navegador y no son enviados a ningún servidor.
